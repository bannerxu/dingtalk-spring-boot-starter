# 工程简介

通过简单配置，完成钉钉机器人消息发送

# 如何接入

```xml

<dependency>
    <groupId>io.github.bannerxu</groupId>
    <artifactId>dingtalk-spring-boot-starter</artifactId>
    <version>1.0</version>
</dependency>
```

    public void waiting() {
        while (!sendTaskQueue.isEmpty()) {
            sleep(2);
        }
        log.info("queue is empty");

    public void sleep(int seconds) {
        try {
